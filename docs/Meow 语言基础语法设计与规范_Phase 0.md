## 1. 欢迎来到 Meow

Meow 是为“最小阻力”设计的高级编程语言。你没学过编程？没关系。你讨厌复杂的符号和规则？那我们来对了地方。Meow 的语法比 Python 更简洁，但功能却对标 Python 与 Java 的完全体。你可以用它写自动化脚本、开发 Web 后端、构建前端界面，甚至编写高性能的编译产物。Meow 最独特的设计是**跨语言互操作**：你可以在 Meow 代码里直接写 Python、JavaScript、SQL 甚至 Rust，用最合适的语言处理每个子任务，而 Meow 负责把它们无缝拼接起来。

本文档将从头开始教你 Meow 的每一个语法点。我们假设你没有任何编程背景，所以每个概念都会先用自然语言解释“这是做什么的”，再给出代码示例。阅读时你会发现，Meow 极其强调“直觉”：省略不必要的符号，统一数据结构，索引从 1 开始，完全兼容中文标点。

## 2. 词法基础

写程序就像写信，**词法**是这封信的书写格式和规定。这一章告诉你 Meow 代码的基本写法：怎么起名字、怎么写注释、用什么符号等等。

### 2.1 字符集
Meow 的源代码文件必须保存为 **UTF-8** 编码。这意味着你可以在注释或字符串里写中文、日文、emoji 等任何文字。这保证了全球开发者都能用自己的语言标注代码。

### 2.2 标点符号双语兼容
编程时最烦的事之一就是来回切换中英文输入法。Meow 彻底解决了这个问题：你完全可以使用中文全角标点写代码，Meow 解释器会自动把它们理解为和英文标点一样的意思。

| 中文标点 | 英文等价款 | 在哪用 |
|---------|------------|--------|
| `，` | `,` | 列表元素、函数参数之间的分隔 |
| `：` | `:` | 字典键值对 |
| `（` `）` | `(` `)` | 改变运算优先级、调用函数 |
| `【` `】` | `[` `]` | 创建列表 |
| `“` `”` | `"` | 包裹字符串 |
| `‘` `’` | `'` | 包裹字符串 |

注意：每条语句必须独占一行，这点不能用逗号改变。

### 2.3 注释
**注释**是写给人看的说明，计算机执行时会完全忽略它们。它用于解释代码的意图、记录待办事项或暂时禁用某段代码。

**单行注释**：从 `#` 开始，到这一行的末尾结束。
```
x = 10  # 这条注释解释 x 的用途
```

**多行注释**：用两个单独的 `#` 符号，把多行内容包在中间。
```
#
这段代码的作者是张三
最后更新于 2026 年 4 月
#
```
这种清晰的包裹方式比传统的 `/* */` 或 `""" """` 更易于阅读和视觉定位。

### 2.4 标识符
**标识符**是你自己给变量、函数等东西取的名字。取名时只有三条规则：
1.  由字母、下划线开头，后面跟上字母、数字、下划线。
2.  区分大小写（`name` 和 `Name` 是两个不同的名字）。
3.  不能与下一节列出的**关键字**重名。

### 2.5 关键字
**关键字**是 Meow 语言内定好的、有特殊用途的单词，你不能把它们用作自己的名字。你可以先把它们当作“语言的内置命令”。

```
if       elif     else     for      while
break    continue def      return
class    init     self     super    interface
implements import   from     as
try      except   finally  raise    error
async    await    match    case
and      or       not      true     false
null     const    enum     extends
```

### 2.6 字面量
**字面量**就是直接在代码里写出来的固定值，它们代表数据本身。

| 类型 | 示例 | 说明 |
|------|------|------|
| 整数 | `10` `-5` `0xFF` `0b1010` | `0xFF` 是十六进制（255），`0b1010` 是二进制（10） |
| 浮点数 | `3.14` `1.5e10` | 带小数点的数字，`1.5e10` 是科学计数法 |
| 布尔值 | `true` `false` | 代表逻辑上的“真”和“假” |
| 空值 | `null` | 表示“没有值”或“不存在” |
| 字符串 | `"hello"` `'hello'` | 一段文本，用单引号或双引号包裹 |

### 2.7 列表：万能的数据容器
在大多数语言里，你有很多容器要学：数组、字典、集合、元组……但在 Meow 里，它们都被统一成了一个东西——**列表**，统一用 `[ ]` 表示。

**它是如何身兼多职的？**

- **当数组用**：元素直接写。
  ```
  numbers = [10, 20, 30]
  ```
- **当字典（键值对）用**：元素写成 `键: 值` 的格式。
  ```
  user = ["name": "Alice", "age": 30]
  ```
- **当不可变数据用**：在列表前加上 `f.`，它就变成了一个“冻结”的列表，谁也无法修改它。
  ```
  point = f.[0, 0]
  ```

### 2.8 字符串内嵌变量
这是一个让你告别繁琐拼接的强大功能：直接在字符串内部嵌入变量的值。你只需在变量名前加上一个 `/` 即可。`/` 符号就像一个魔法棒，告诉 Meow：“去把这个变量的值放在这里”。
```
name = "世界"
print "你好 /name"          # 输出: 你好 世界
```
如果要嵌入的是一个复杂的表达式，记得用 `()` 把它包起来。
```
print "1 + 1 = /(1 + 1)"  # 输出: 1 + 1 = 2
```
当变量名后面紧跟着其他普通字符，Meow 采用最长词法匹配规则识别变量名（匹配 `[a-zA-Z_][a-zA-Z0-9_]*`）。若需明确边界，请使用 `/(expr)` 形式，详见 2.11 节。

### 2.9 运算符
**运算符**是对数据进行操作的符号。Meow 提供了丰富的运算符。

| 类别 | 符号 | 说明 |
|------|------|------|
| 算术 | `+ - * / // % **` | `//` 是整除，`%` 是取余数，`**` 是乘方 |
| 比较 | `== != > < >= <=` | 用来判断大小或相等关系，结果是一个布尔值 |
| 逻辑 | `and or not` | 用于组合或反转多个布尔条件 |
| 赋值 | `= += -= *= /=` | `a += 1` 就是 `a = a + 1` 的简写 |
| 成员 | `in not in` | 检查一个元素是否存在于列表中 |
| 列表运算 | `++ && -- --/` | 专门用于列表的集合运算，参见下表 |
| 匿名函数 | `--` | 在一个短小的函数中，用来隔开参数和函数体 |

**列表运算符详解** (设 `a = [1, 2, 3]`, `b = [2, 3, 4]`)：

| 符号 | 含义 | 示例 | 结果 |
|------|------|------|------|
| `+` | 简单拼接 | `a + b` | `[1,2,3,2,3,4]` |
| `++` | 并集（去重） | `a ++ b` | `[1,2,3,4]` |
| `&&` | 交集（找共同元素） | `a && b` | `[2,3]` |
| `--` | 差集（a有b无） | `a -- b` | `[1]` |
| `--/`| 对称差（找出只在一个列表中出现的元素） | `a --/ b` | `[1,4]` |

### 2.10 索引、语句与缩进

- **语句**：Meow 里一行就是一条独立的指令。不像有些语言需要用分号 `;` 来结束，你只需要换行即可。
- **索引**：为了符合人类数数的习惯，Meow 中的位置索引（Index）**从 1 开始**。访问一个序列的最后一个元素用 `-1`，倒数第二个用 `-2`，以此类推。
  ```
  s = "你好世界"
  s[1]    # 第一个字: "你"
  s[-1]   # 最后一个字: "界"
  ```
- **缩进**：Meow 使用 **4 个空格**来区分代码的“层级”或“块”。这就像写文章时分段，层层递进，视觉上非常清晰。
  ```
  if score > 90
      print "优秀"      # 这行前面有 4 个空格，属于 if 内部
      print "继续加油"   # 这行也属于 if 内部
  print "检查结束"      # 这行顶格写，不属于 if，一定会执行
  ```

### 2.11 无括号调用的参数限制

Meow 支持无括号函数调用（如 `print "Hello"`），但为避免与表达式运算符产生二义性，调用参数的起始 token 有以下限制：

**合法的参数起始 token：**
- 字面量：字符串 `"..."`、数字 `123`、布尔 `true/false`、空值 `null`
- 标识符：变量名 `x`、关键字 `self`
- 复合表达式：`(` 包裹的表达式
- 列表/字典：`[` 起始的字面量
- 匿名函数：`fn` 关键字
- 一元负号：**不在合法起始列表中**（会被优先判定为二元减法运算符）

**不合法的参数起始 token（将触发二元运算符解析）：**
- `-`（一元/二元负号）—— 请使用 `print (-5)` 替代 `print -5`
- `not`（逻辑非）—— 请使用 `print (not x)` 替代 `print not x`

**字符串插值变量边界：**
- 内嵌简单变量采用最长词法匹配规则，匹配模式为 `[a-zA-Z_][a-zA-Z0-9_]*`
- 若变量名后紧跟普通字符，需用 `/(expr)` 形式明确边界
  - ✅ `"hello /(name)world"` → 明确边界
  - ✅ `"hello /name world"` → 空格自然分隔
  - ⚠️ `"hello /nameworld"` → 视为查找名为 `nameworld` 的变量

## 3. 数据类型与基本操作

变量是用来装数据的“盒子”。**数据类型**就是告诉程序这些盒子分别装的是什么种类的“东西”。虽然 Meow 是动态类型语言（不用写类型名），但了解数据类型依然是编程的基础。

### 3.1 数字
计算机天生就是为计算而生的。Meow 中的数字和数学里的用法基本一致。
```
a = 10 + 5       # 15
b = 10 / 3       # 3.333... (常规除法)
c = 10 // 3      # 3 (整除，只保留整数部分)
d = 10 % 3       # 1 (求模，获取余数)
e = 2 ** 3       # 8 (2 的 3 次方)
```
如果需要把一个字符串转换成数字，只需使用 `int` 或 `float` 函数即可。
```
num = int "100"          # 字符串"100"变成了数字 100
pi = float "3.14"        # 字符串"3.14"变成了数字 3.14
```

### 3.2 字符串
字符串就是“一段话”。Meow 让你能非常轻松地处理它。所有操作都是 `字符串.功能名` 的形式，简单直观。

| 我想做什么 | 代码写法 | 例子 | 结果 |
|----------|----------|------|------|
| 求长度 | `s.length` | `"abc".length` | `3` |
| 转大写 | `s.up` | `"hi".up` | `"HI"` |
| 转小写 | `s.low` | `"HI".low` | `"hi"` |
| 去空格 | `s.strip` | `" a ".strip` | `"a"` |
| 替换文字 | `s.replace 旧, 新` | `"aba".replace "a", "x"` | `"xbx"` |
| 分割 | `s.split 分隔符` | `"a,b".split ","` | `["a","b"]` |
| 是否包含 | `s.contains 子串` | `"hello".contains "ell"` | `true` |

### 3.3 布尔与空值
布尔值只用来做判断，只有 `true` 和 `false` 两种情况。`null` 代表“空”或“啥也没有”。在需要做判断的地方，以下这些值都会被当作 `false` 看待：`false`, `null`, `0`, `""`, `[]`。其他所有值都会被当作 `true`。

### 3.4 玩转列表
列表是 Meow 的头号工具，你需要熟练掌握它。

**增删改查四部曲：**

| 我想做什么 | 代码写法 | 示例 |
|----------|----------|------|
| 读取第1个 | `list[1]` | `nums[1]` |
| 修改 | `list[1] = 新值` | `nums[1] = 100` |
| 末尾追加 | `list add 新元素` | `nums add 99` |
| 按索引删 | `list[n] del` | `nums[2] del` |
| 按值删 | `list remove 值` | `nums remove 100` |
| 字典取值 | `list["键名"]` | `user["name"]` |
| 字典赋值 | `list["键名"] = 新值` | `user["age"] = 31` |

**列表的更多玩法：**
- `list.length` —— 告诉你列表里有多少个东西
- `list.unique` —— 自动去掉重复的元素，返回一个新列表
- `list.key` / `list.value` —— 对于键值对列表，分别拿出所有“键”或所有“值”

**列表推导**是创建列表的一种优雅速写，它能从一个旧列表快速生成新列表：
```
# 生成 1 到 10 的平方数列表
squares = [x * x for x in range 1, 11]
```
这段代码的意思是：对于从 1 到 10 的每个数 x，都计算 x * x，然后把这些结果装进一个新列表里。

## 4. 变量与作用域

- **变量**：你给一个数据起的名字。在 Meow 里，不用提前声明类型，直接 `name = "Alice"` 就行。
- **常量**：一旦定义就不能再修改的值，用 `const` 来声明。例如 `const PI = 3.14159`。
- **作用域**：变量在哪个代码块（即哪层缩进）里被创建，它的“寿命”就局限在这个代码块里。这叫块级作用域。
  ```
  x = 10
  if true
      y = 20    # y 在这里创建
  print y       # 错误！ y 在 if 块外面已经不存在了
  ```
  这个规则保证你的程序结构清晰，不会因为同名变量在不同地方互相干扰。

## 5. 控制流

程序不会总是从上到下简单执行。**控制流**就是用来指挥程序在不同条件下“拐弯”或“转圈”的。

### 5.1 条件判断
让程序根据不同的情况做出不同的反应。关键字是 `if`、`elif`（表示“否则如果”）、`else`。
```
score = 85

if score >= 90
    print "优秀"        # 只有 >=90 才走这里
elif score >= 60
    print "及格"        # 不满足 >=90 但 >=60，走这里
else
    print "不及格"       # 以上都不满足，走这里
```

### 5.2 循环
让程序反复执行一段代码。

- **`for` 循环**：用于遍历一个序列，比如一个列表或一串数字。
  ```
  for fruit in ["苹果", "香蕉"]
      print fruit
  ```
- **`while` 循环**：只要条件为真，就不断执行。
  ```
  count = 1
  while count <= 5
      print count
      count += 1   # count 自身加 1，直到不再 <= 5
  ```
- **`break` 与 `continue`**：在循环中，`break` 是立刻跳出整个循环，`continue` 是跳过本次循环剩下的部分，直接开始下一次循环。

## 6. 函数

函数是一段**可重复使用的代码块**，给它一个名字和一些参数，它就能为你完成特定任务。

**定义与调用：**
```
# 定义: 一个叫 greet 的函数，需要一个参数 name
def greet name
    print "你好, /name"

# 调用: 让函数跑起来
greet "小明"
```

**返回值：** 让函数计算完把结果“交出来”。
```
def add a, b
    return a + b

result = add 3, 5    # result 现在是 8
```

**匿名函数：** 一些小功能，没必要特意取个名字，可以当场定义。用 `fn` 开头，参数和表达式之间用 `--` 隔开。
```
# 定义一个临时函数，让数翻倍
double = fn x -- x * 2
double 5    # 结果是 10
```

## 7. 类与对象

**类**是真实世界事物的蓝图，**对象**是根据这个蓝图造出来的具体实例。比如，“学生”是一个类，而“张三”这个具体学生就是“学生”类的一个对象。

**定义类与构造方法：**
`init` 是一个特殊的方法，叫“构造方法”，它规定了创建一个新对象时需要填什么参数。
```
class Student
    def init self, name, score
        self.name = name
        self.score = score

    def show self
        print "/self.name 考了 /self.score 分"
```

**创建并使用对象：**
```
s1 = Student "张三", 95   # 创建一个叫张三的学生对象
s1.show                    # 调用这个对象的 show 方法
```

**继承：** 在已有类的基础上，创建一个更具体的类。
```
# 研究生也是学生，所以它"继承"学生
class GraduateStudent extends Student
    def show self            # 研究生有自己独特的自我介绍方式
        print "我, 研究生 /self.name"
```

**接口：** 接口是一份“合同”，它规定了一个类**必须**提供哪些方法，但不提供具体实现。
```
interface Flyable
    def fly self

# 飞机实现了 Flyable 这个接口，所以它必须有 fly 方法
class Airplane implements Flyable
    def fly self
        print "引擎飞行"
```

## 8. 异常处理

程序运行时免不了出错，比如要打开的文件不存在，或者网络突然断了。如果不处理，程序会直接崩溃。**异常处理**就是让程序优雅地应对这些意外状况。

```
try
    # 把可能会出错的代码放在这里
    result = 10 / 0
except
    # 如果上面的代码出错了，就执行这里的补救措施
    print "出错啦！不能除以零"
finally
    # 无论出不出错，这里面的代码都一定会执行
    print "计算结束"
```

你还可以自己创造新的错误类型，让代码的错误处理更有层次感。
```
# 定义一个名叫 MyError 的异常
error MyError "发生了一个自定义错误"

# 在你的代码中抛出这个异常
if true
    raise MyError "数据验证失败"
```

## 9. 模块化编程

当代码变多时，把所有东西塞在一个文件里会非常混乱。**模块化**就是让你把代码拆分到不同的 `.meow` 文件中，每个文件就是一个模块，里面可以导出函数和类，供其他文件使用。

```
# 文件: my_math.meow
def add a, b
    return a + b

# 文件: main.meow
import my_math              # 导入那个文件，my_math 就是文件名
print my_math.add 3, 5     # 使用模块里的函数
```

你也可以只从一个模块里拿你需要的东西：
```
from my_math import add
print add 3, 5
```

## 10. 跨语言互操作

这是 Meow 的“王牌”特性。它允许你在 Meow 代码中，**直接嵌入其他编程语言的代码块**，无缝调用它们庞大的生态库。

**基本语法：**
```
import 语言名
{
    这里是符合目标语言语法的代码
}.语言名
```
语言后缀不区分大小写，例如 `.python`, `.js`, `.ts`, `.java`, `.rs`, `.go`, `.sql`, `.html` 等。

**导入外部语言的包：**

除了嵌入代码块，你还可以直接导入某个语言生态中的现成包，并在后续代码中以 `包名.语言名` 的形式调用它。

```
import 包名.语言名
```

```
// 导入 Python 的 numpy 包
import numpy.python

data = numpy.python.array [[1, 2], [3, 4]]
result = numpy.python.dot data, data.T
print result

// 导入 JavaScript 的 lodash
import lodash.js

chunks = lodash.js.chunk ["a", "b", "c", "d"], 2
```

导入后，该包的 API 会挂在 `包名.语言名` 命名空间下，调用方式与目标语言的原始 API 保持一致。这比编写内联代码块更简洁，适合直接使用成熟第三方库的场景。

**现实世界示例：**

你想在 Meow 里画个图表，但 Meow 没有图表库。没关系，我们用 Python 的 `matplotlib`！
```
import python
{
import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 1])
plt.show()
}.python
```
现在你想用 JavaScript 处理一段前端逻辑：
```
import js
{
const _ = require('lodash');
const result = _.chunk(['a', 'b', 'c', 'd'], 2);
}.js
```
甚至，你可以把外部语言的功能，包装成一个 Meow 函数！让它看起来和原生 Meow 代码一模一样。
```
# 这是一个 Meow 函数，但内部是 Python 代码
def py_sort arr
    import python
    {
    return sorted(arr)
    }.python

# 调用这个 Meow 函数
sorted_list = py_sort [3, 1, 2]
print sorted_list   # 输出: [1, 2, 3]
```

## 11. 异步编程

很多操作很耗时，比如从网上下载数据，如果傻等会卡住整个程序。**异步编程**让你在等待过程中，程序还能干别的事。

`async` 标记一个函数是异步的，`await` 用来等待一个异步操作完成，而 `all` 可以同时开启多个异步任务。
```
async def fetch_all_data
    # 三个网络请求同时发出，不必一个一个排队等！
    results = await all [get_user(1), get_posts(), get_comments()]
    return results
```

## 12. 枚举与模式匹配

**枚举**就是将一组相关的常量组织在一起，让代码更具可读性。
```
enum Status
    ACTIVE
    INACTIVE
    BANNED
```

**模式匹配**可以看作是“超级强化版的 if-else”。它用 `match` / `case` 语法，能根据数据的结构、值去进行不同的处理，非常强大。
```
match user
case ["name": n, "age": a]
    print "找到用户 /n, 年龄 /a"
case ["error": err]
    print "出错了: /err"
else
    print "未知格式"
```

## 13. 装饰器与元编程

**装饰器**允许你在函数执行前后自动“织入”额外的逻辑，比如日志记录、权限校验。它用到 `@` 符号。
```
# 一个打印日志的装饰器
def log func
    def wrapper args...
        print "函数开始执行"
        result = func args...
        print "函数执行完毕"
        return result
    return wrapper

@log
def say hi name
    print "你好 /name"

# 调用 say hi 时，会自动打印出开始和结束的日志
```

## 14. 泛型

**泛型**让你可以写出不依赖具体类型的“通用”代码，同时保持类型安全。比如，一个处理“箱子”的类，可以装整数，也可以装字符串，逻辑是一样的。
```
class Box<T>
    def init self, item: T
        self.item = item

    def get self = T
        return self.item

num_box = Box<int> 100
str_box = Box<str> "meow"
```

## 16. 代码风格指南

好的代码风格就像一手好字，能让他人（包括未来的你）更容易读懂你的程序。Meow 社区推荐以下约定：

- **文件后缀**：`.meow`
- **主入口**：每个程序的启动逻辑，约定放在 `main` 函数中。
  ```
  def main
      print "Meow 启动！"

  if __name__ == "__main__"
      main
  ```
- **命名**：
  - 变量与函数：`snake_case`，如 `user_name`, `get_data`
  - 类、接口、枚举：`PascalCase`，如 `UserService`, `Drawable`
  - 常量：`UPPER_CASE`，如 `MAX_SIZE`
- **每行一条语句**。
- **操作符两边用空格隔开**：`a = 1 + 2`。
